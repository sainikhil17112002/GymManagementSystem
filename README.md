# Gym Management System

A modern Gym Management System built with **Django** to simplify the daily operations of fitness centers. The application provides an intuitive dashboard for managing members, memberships, attendance, products, inventory, and more.

## Features

* 📊 Dashboard with analytics
* 👥 Member Management
* 💳 Membership Plans
* 📅 Attendance Tracking
* 📈 Member Growth & Analytics
* 🛒 Products Management
* 📦 Inventory Management
* 🧾 Reception Panel
* 📱 Responsive Admin Interface

## Technologies Used

* Python 3
* Django
* HTML5
* CSS3
* Bootstrap
* JavaScript
* SQLite (Default)

## Installation

1. Clone the repository:

   ```bash
   git clone https://github.com/YOUR_USERNAME/GymManagementSystem.git
   ```

2. Navigate to the project:

   ```bash
   cd GymManagementSystem
   ```

3. Create a virtual environment:

   ```bash
   python -m venv venv
   ```

4. Activate the virtual environment:

   **Windows**

   ```bash
   venv\Scripts\activate
   ```

   **Linux/macOS**

   ```bash
   source venv/bin/activate
   ```

5. Install dependencies:

   ```bash
   pip install -r requirements.txt
   ```

6. Apply migrations:

   ```bash
   python manage.py migrate
   ```

7. Run the development server:

   ```bash
   python manage.py runserver
   ```

8. Open your browser:

   ```
   http://127.0.0.1:8000/
   ```
